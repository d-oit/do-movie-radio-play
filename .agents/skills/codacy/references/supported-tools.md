# Supported Codacy Tools

Codacy supports hundreds of tools. For this repository, the following are most relevant:

## Core Tools

| Tool | Language | Status |
|------|----------|--------|
| Rust / Clippy | Rust | ✅ Supported (Cloud) |
| ShellCheck | Shell | ✅ Supported |
| markdownlint | Markdown | ✅ Supported |

## Local Analysis CLI Support

| Tool | Status |
|------|--------|
| ShellCheck | ✅ Supported |
| markdownlint | ✅ Supported |
| Clippy | ⚠️ Dependent on local Rust environment |

Always use `codacy pull-request` to see the authoritative list of issues from the Cloud analysis.
